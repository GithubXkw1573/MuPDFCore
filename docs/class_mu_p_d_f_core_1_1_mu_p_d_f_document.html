Create a new MuPDFDocument from an array of bytes.

Parameters
contextThe context that will own this document.
dataAn array containing the data bytes that make up the document. This must not be altered until after the MuPDFDocument has been disposed! The address of the array will be pinned, which may cause degradation in the Garbage Collector's performance, and is thus only advised for short-lived documents. To avoid this issue, marshal the bytes to unmanaged memory and use one of the IntPtr constructors.
fileTypeThe type of the document to read.
